Der TL431 ist eine Referenz, die mit Hilfe eines 71,5K-Widerstands und eines Potentiometers auf den konstanten Stromreglermodus eingestellt ist. Diese Referenz wird mit der Sensorspannung von der Ausgangsseite des Widerstands verglichen, um den Strom zu begrenzen.  Die Schaltung unten ist die TL431-Schaltung, die eine konstante Stromquelle für die Operationsverstärker bereitstellt.
